# OP JAK Návraty DMP

This data management plan (DMP) template is a modified version of the [MŠMT DMP](https://registry.fair-wizard.com/document-templates/muni:msmt-dmp:1.2.0) developed to be used for project
calls of the [Ministry of Education Youth and Sport](https://msmt.gov.cz/?lang=2). To comply with the OP JAK Návraty's requirements regarding logotype, this document template keeps the exactly same structure as MŠMT DMP, but has an updated version of the MŠMT logo on the front page and in the header. The footer was also corrected to attribute FAIR Wizard instead of Data Stewardship Wizard.

It can be used with the [Data Stewardship Wizard (DSW)](https://ds-wizard.org/) and [FAIR Wizard](https://fair-wizard.com/) DMP tools.


## Usage
As of now, this template is available through a[ GitLab repository](https://gitlab.ics.muni.cz/fr-cesnet-756-2024/dsw-document-template-msmt.git).
